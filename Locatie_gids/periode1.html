Webontwikkeling Basics

HTML

In periode 1 hadden we het over HTML. HTML was het eerste dat we leerden bij programmeren en coderen, omdat dat de basis is.

CSS

CSS (Cascading Style Sheets) hadden we ongeveer een week of twee na HTML. Het was de tweede belangrijkste taal bij coderen. Je kunt het vergelijken met een mens: HTML is het skelet en CSS is het vlees. Zonder beide kun je geen goede website maken.

JavaScript

JavaScript kregen we ongeveer anderhalf maand na de eerste HTML-les. JavaScript is net als CSS, vind ik persoonlijk. Het is ongeveer hetzelfde, alleen is JavaScript een stuk makkelijker, vind ik zelf.
